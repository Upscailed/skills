# UP/Skills

## Wat is dit?

Skills-repo van Upscailed: collectie skills die Upscailed bouwt en publiceert, zowel voor eigen workflow als voor klanten. Gestart op 2026-05-05 als publieke skills-repo (GitHub `Upscailed/skills`). Inhoudelijke skills wonen ook in de Upscailed-kluis op Drive (`Upscailed-OS/IT/Skills/`); deze repo is de versie voor publicatie/distributie.

> Verdere positionering en distributie-strategie nog aan te vullen door Iwan.

## Werkmethode voor agents

Tasks/acties voor skills-publicatie staan in de Iwan OS vault per scope; Mission Control (`localhost:3333`) bundelt alles. Werk lokaal — geen externe API/MCP.
